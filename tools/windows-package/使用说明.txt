申硕英语词汇学习 · Windows 离线版
============================================

一、怎么打开
--------------------------------------------
1. 把整个压缩包解压到任意文件夹（桌面、D 盘都行），不要只解压单个文件
2. 双击「启动学习软件.bat」
3. 等几秒，浏览器会自动打开 http://127.0.0.1:5173/，看到词汇学习界面就成功了

二、为什么会弹出一个黑色的命令行窗口
--------------------------------------------
那是软件自己的"小服务器"，所有词汇数据都在本地、不联网。
学习的时候请保持这个黑窗口开着；关掉它 = 关掉学习软件。
学完了直接把黑窗口和浏览器都关掉就行，不会有后台残留。

三、首次启动可能会遇到的提示
--------------------------------------------
- 如果 Windows 弹「Windows 已保护你的电脑」蓝色警告：
    点「更多信息」→「仍要运行」即可
    （这是给陌生 .bat 文件的默认提醒，跟病毒无关）

- 如果浏览器没自动打开：
    手动打开 Chrome 或 Edge，地址栏输入 http://127.0.0.1:5173/

- 如果提示「端口 5173 被占用」：
    说明你之前已经开过一次，找到那个黑窗口先关掉

四、数据存哪里
--------------------------------------------
- 词库（5390 个词）整个打包在 web/data/vocab.json，不联网
- 你的学习进度（认识了哪些词、薄弱词、笔记、收藏等）保存在
  浏览器自己的 localStorage 里。
- 想清空进度：用过的浏览器里，打开 http://127.0.0.1:5173/，
  右下角"词库概况"卡里点「🗑 清除所有学习进度」

五、给原始作者反馈或升级
--------------------------------------------
- 这个软件由 Codex / Claude Code 帮我从大纲 PDF 一步步搭起来的
- 如果想升级版本：找我（朋友）要新的压缩包，把旧版整个文件夹删掉重新解压即可，
  不会丢学习进度（数据存在浏览器里，不在这个文件夹里）

六、推荐学习节奏（高考 76 分基础）
--------------------------------------------
打开后默认进入「高考基础」阶段（2888 词），右侧"难度梯度"面板
可以点开看更细的「第 1 段 / 第 2 段 ...」(每段 300 词)。

阶段顺序建议：
    1. 高考基础 (2888) → 大约 3 个月，把高考词彻底打牢
    2. 四级核心 (1060) → 约 5 周
    3. 六级提升 (1220) → 约 6 周
    4. 申硕进阶 (98)  → 几天
    5. 拔高识记 (124) → 看脸熟即可

每天 30 个新词比较稳，加上复习量持续 5-6 个月能完整过一遍。

七、快捷键
--------------------------------------------
    ← →   翻页
    1-4   选择 A/B/C/D 选项
    K/U/F  认识 / 有点印象 / 不认识
    空格  显示/收起释义
    P     播放发音
    S     收藏
    /     跳到搜索框
